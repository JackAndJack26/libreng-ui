# LibreNG UI

LibreNG UI is a rich set of open source (MIT) UI components for Angular 22+. It is a community-maintained fork of the MIT-licensed PrimeNG v21 by PrimeTek, with unchanged component APIs and selectors.

## Installation

```bash
npm install @libreng/ui
```

## Usage

```ts
import { ButtonModule } from '@libreng/ui/button';
```

```html
<p-button label="Submit" />
```
